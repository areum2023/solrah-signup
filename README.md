# solrah-signup

간단한 정적 React 데모를 빠르게 확인할 수 있도록 CDN으로 React, Babel, Tailwind를 불러오는 구조입니다.

## 실행 방법
1. 프로젝트 루트에서 간단한 웹서버를 실행합니다.
   ```bash
   python -m http.server 4173
   ```
2. 브라우저에서 http://localhost:4173 로 접속합니다.
3. 네비게이션 하단 탭(홈/시설예약/커뮤니티/마이)과 사이드바 버튼을 눌러 화면 상태를 확인할 수 있습니다.

별도의 빌드나 의존성 설치가 필요하지 않습니다.
